Zatree简介
==================================

当zabbix走进千家万户的时候，我们也许需要做点什么。

zatree是监控软件zabbix的一个插件，主要功能是提供host group的树形展示和在item里指定关键字查询及数据排序。

安装方法进到不同版本目录看readme
